1Point2.fish

A web app to check for new releases of various software packages.

1Point2.fish maintains a list of popular software (mostly web development related) and performs a nightly check on their download pages to see if a new version is available. If it is, the app sends notification emails to whomever has subscribed to monitor that particular package.
jQuery, PHP5, MariaDB, EC2 administration, CRONTab, cURL
