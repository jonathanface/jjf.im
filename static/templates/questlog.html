QuestLog

A web forum for online roleplaying game enthusiasts.

This project was largely a modernization of an old website from 2002. It was previously written in PHP4 and was dependent upon synchronous page loading.

I rewrote the back-end in Go, and added a RESTful API to update, edit, and delete user posts. I also added support for things like dice rolling via text input, which required heavy use of regular expression matching.

It is still under development at this time.
jQuery, GoLANG, MariaDB, EC2 administration
